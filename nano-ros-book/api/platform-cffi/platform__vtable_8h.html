Detailed Description

nros platform abstraction — C vtable for porters writing a platform layer in C (or any C-ABI language).

AUTO-GENERATED by cbindgen from nros-platform-cffi/src/lib.rs — DO NOT EDIT MANUALLY.

A platform implementor:

  1. defines a nros_platform_vtable_t global with every function pointer filled in (see field-level docs for return-value conventions),
  2. calls nros_platform_cffi_register(&vtable) before opening a session,
  3. links the resulting object against the nros static library built with --features platform-cffi.
