nros codegen — build-tool-facing C/C++ binding generation.

Phase 195.A: folds the former standalone nros-codegen binary (nros-codegen-c) into the canonical nros CLI. Same engine (cargo_nano_ros), same call shape, so the cmake / build.rs consumers only change the program name (nros-codegen … → nros codegen …):

nros codegen –args-file [–language c|cpp] [–verbose] nros codegen resolve-deps –package-xml –output-cmake [–verbose]

Distinct from nros generate (the user-facing, package.xml-driven surface): this is the JSON---args-file contract the build system already speaks.
