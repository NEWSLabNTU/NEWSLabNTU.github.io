W4a — same as install_node_typed but seeds the node’s NodeContext with the launch-baked <param> initial values, so a register/init-time ctx.param(name) observes the compile-time launch value (RFC-0004 §10). params must outlive the call.

§Safety

executor must be the live *mut Executor handle a typed entry passes, valid for the call.
