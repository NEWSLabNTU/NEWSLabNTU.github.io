Expand description

The executable counterpart of Component (W.5.1).

register (declarative) stays the planning SSOT; this binds runnable bodies. The generated runtime builds State once via init, then routes every fired callback to on_callback. Trait-dispatch (no boxed dyn, no alloc) keeps it no_std.
