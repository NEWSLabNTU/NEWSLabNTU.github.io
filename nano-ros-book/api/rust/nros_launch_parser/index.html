Phase 212.N.11 — ROS 2 launch.xml parser (v1 tag set).

Copy-paste compatibility with stock nav2 / Autoware / turtlebot3 *.launch.xml files. Per design-doc §11.5 the v1 tag set is:

  • <launch> — root.
  • <arg name=".." default=".." value=".."/> — launch arg.
  • <node pkg=".." exec=".." name=".." namespace=".."/> — spawn.
  • <param name=".." value=".."/> — child of <node>.
  • <remap from=".." to=".."/> — child of <node> or <group>.
  • <group ns=".."/> — namespace wrapper, may carry <node>, <include>, <remap>.
  • <include file=".."/> — recursive XML pull; optional child <arg> pass-through.

Substitutions:

  • $(find <pkg>) — pkg-index lookup via PkgIndex.
  • $(var <arg>) — current launch-arg scope.
  • $(env <name>) — std::env::var(name).

Nested substitutions are NOT supported in v1 ($(find $(var pkg)) errors). Python .launch.py is out of scope.
