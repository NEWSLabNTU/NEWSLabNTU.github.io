1//! nros-rmw-zenoh: Zenoh-pico RMW backend for nros
2//!
3//! This crate provides the zenoh-pico transport implementation,
4//! combining the safe Rust API over zenoh-pico FFI with the
5//! transport layer that implements nros-rmw traits.
6//!
7//! # Platform Backends
8//!
9//! Select one backend via feature flags:
10//! - `platform-posix` - Uses POSIX threads, for desktop testing
11//! - `platform-zephyr` - Uses Zephyr RTOS threads
12//! - `platform-bare-metal` - Uses polling (bare-metal platforms)
13//! - `platform-freertos` - Uses FreeRTOS threads + lwIP sockets
14//! - `platform-threadx` - Uses ThreadX threads + NetX Duo sockets
15
