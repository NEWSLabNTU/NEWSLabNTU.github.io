Expand description

Runtime registrar installed by a backend that builds per-type descriptors (e.g. Cyclone DDS).

Receives the message type’s flattened schema — its ROS type name and its &'static field table — and registers/builds the backend’s descriptor for it. Returns Err(()) on any backend-side failure (registry full, descriptor build error); the core maps that onto TransportError::PublisherCreationFailed.

Both arguments are &'static so the registrar may cache them by reference in a bounded 'static registry without copying.
