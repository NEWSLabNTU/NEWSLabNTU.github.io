POSIX serial (PTY / UART) transport via termios.

Implements PlatformSerial on PosixPlatform. The handle is a libc file descriptor — multiple concurrent devices are supported from a single platform impl (open returns the FD; every other method takes it back). Single-session consumers like nros-rmw-xrce::platform_serial stash the FD in a local static between init_platform_serial_transport and the transport callbacks.
