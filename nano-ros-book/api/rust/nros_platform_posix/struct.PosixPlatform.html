pub fn udp_listen( sock: *mut c_void, endpoint: *const c_void, timeout_ms: u32, ) -> i8

Phase 71.21 — bind a UDP socket for inbound use.

Differs from udp_open in that this calls bind(2) after socket(2). udp_open is for zenoh-pico-style outbound-only usage where the local port is ephemeral and never observed by peers; DDS needs the local port to be the deterministic RTPS PSM port so peers can address replies back.
