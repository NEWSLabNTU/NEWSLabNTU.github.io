pub fn emit_run_plan(plan: &NrosPlan) -> String
Expand description

Render the body of $OUT_DIR/run_plan.rs for the given plan.

Returns a UTF-8 string the caller writes verbatim. Doc-comments in the emitted code spell out the lineage (planner version, system name, instance count) so a curious user can introspect without re-running codegen.
