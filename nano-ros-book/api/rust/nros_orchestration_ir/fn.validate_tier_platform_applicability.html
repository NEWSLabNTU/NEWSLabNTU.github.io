Reject tier knobs the SELECTED target cannot honor — a bake-time error, never a silent ignore (the domain-0 lesson). Shared by the CLI codegen path and the nros::main! proc-macro so both bakes agree.

Rules (RFC-0052 rejection table):

  • preempt_threshold in the selected sub-table → ThreadX only.
  • sched_class in the selected sub-table → POSIX-family targets only (posix/native/nuttx).
  • class = "interrupt" → rejected (v1).
  • class = "time_triggered" without period_us → rejected.
